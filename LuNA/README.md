## 使用方法：
### 安卓
将Scene.pck、__g00_patch_001.pck复制到“Android\data\jp.co.product.kn.lunaria\files\”内替换。  

### 越狱iOS
将Scene.pck、__g00_patch_001.pck复制到“程序(用户)\LUNARiA\kn0020_kn_LUNARiA.app\”内替换。  

**注意：**
 * 安卓需要先正常下载完游戏数据后再替换！  
 * 只替换这两个文件，其他文件是特殊情况使用的，不要替换！  

## 如果安卓无法进入data文件夹：
